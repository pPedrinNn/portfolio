Aprendizados

O Projeto Integrador I proporcionou uma experiência prática fundamental na área de desenvolvimento de sistemas. A principal ênfase foi no uso de ferramentas como Git e GitHub para controle de versão e colaboração.

Entendi a importância da organização de versões, de trabalhar com branchs para evitar conflitos e como manter um repositório limpo e funcional em equipe. A prática com comandos como git add, git commit, git push e git pull foi constante.

Habilidades Desenvolvidas

  • Uso de Git e GitHub no desenvolvimento
  • Criação e gerenciamento de branchs
  • Versionamento de código em equipe
  • Organização e documentação de projetos
  • Compreensão de fluxo de trabalho colaborativo
