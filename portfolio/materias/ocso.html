Unidade Central de Processamento (CPU)

A CPU é o cérebro do computador, responsável por executar as instruções de um programa, realizando operações aritméticas, lógicas, de controle e de entrada/saída.

Componentes Principais:

  • Unidade de Controle (UC): Interpreta instruções e gera sinais de controle
  • Unidade Lógica e Aritmética (ULA): Realiza operações matemáticas e lógicas
  • Registradores: Armazenam dados e instruções temporariamente
  • Memória Cache: Armazena dados frequentemente acessados

Ciclo de Instrução:

  1. Busca (Fetch): Busca a instrução na memória
  2. Decodificação (Decode): Interpreta a instrução
  3. Execução (Execute): Executa a operação
  4. Armazenamento (Store): Guarda o resultado

Hierarquia de Memória

Os computadores possuem uma hierarquia de memória que varia em velocidade, capacidade e custo:

  • Registradores: Mais rápidos, menor capacidade, dentro da CPU
  • Cache (L1, L2, L3): Alta velocidade, pequena capacidade
  • Memória Principal (RAM): Volátil, acesso rápido
  • Armazenamento Secundário (HDD/SSD): Não volátil, maior capacidade

Técnicas de Pipeline

Técnica que permite a execução sobreposta de instruções, dividindo o processamento em estágios:

  1. Busca de Instrução (Fetch)
  2. Decodificação (Decode)
  3. Execução (Execute)
  4. Acesso à Memória
  5. Escrita de Resultado

Vantagens: Aumento de desempenho e melhor utilização de recursos

Desafios: Hazards (conflitos de dados, controle e estruturais)

Tópicos Específicos

Unidade Lógica e Aritmética (ULA)

Funções: Realiza operações aritméticas (adição, subtração) e lógicas (AND, OR, NOT, XOR)

Operações: Trabalha com dados binários, usando complemento para dois

História: Conceito proposto por John von Neumann em 1945

Exemplo de Operações:

Operações Aritméticas: Soma, Subtração, Multiplicação, Divisão
Operações Lógicas: AND, OR, NOT, XOR
Operações de Deslocamento: Shift left, Shift right

Barramentos

Canais de comunicação entre componentes do computador:

  • Barramento de Dados: Transfere dados entre componentes
  • Barramento de Endereços: Especifica locais de memória
  • Barramento de Controle: Coordena operações

Tipos de Barramentos:

  • PCI: Para periféricos internos
  • PCI Express: Versão mais rápida do PCI
  • USB: Para periféricos externos
  • SATA: Para dispositivos de armazenamento

Arquiteturas CISC vs RISC

CISC

  • Instruções complexas
  • Microcódigo
  • Tamanho variável de instruções
  • Exemplo: Intel x86

RISC

  • Instruções simples
  • Execução em ciclo único
  • Tamanho fixo de instruções
  • Exemplo: ARM
