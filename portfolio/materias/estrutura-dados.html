Aprendizados

Esta matéria me proporcionou um entendimento profundo sobre como organizar e manipular dados de forma eficiente em programas. Aprendi a importância de escolher a estrutura de dados adequada para cada tipo de problema e como implementar essas estruturas na linguagem C.

Os projetos práticos foram essenciais para consolidar o conhecimento teórico, especialmente na parte de alocação dinâmica de memória e manipulação de ponteiros, que são fundamentais para trabalhar com estruturas de dados dinâmicas.

Detalhes do Conteúdo

1. Vetores (Arrays)

Conceito: Um vetor é uma estrutura linear que armazena elementos do mesmo tipo em posições consecutivas de memória.

  • Tipos: Vetores não ordenados e ordenados
  • Operações: Inserção, pesquisa linear/binária, exclusão
  • Algoritmos: Bubble Sort, Selection Sort, QuickSort
  • Implementações: Array estático e dinâmico

2. Listas Encadeadas

Conceito: Estrutura dinâmica onde cada nó contém um valor e um ponteiro para o próximo nó.

  • Tipos: Simples, duplamente encadeada, circular
  • Operações: Inserção no início/fim, remoção, busca, percurso
  • Vantagens: Tamanho flexível, inserções/remoções eficientes

3. Pilhas e Filas

Pilhas (LIFO)

  • Operações: push, pop, peek

Filas (FIFO)

  • Tipos: Simples, com prioridade, circular
  • Operações: enqueue, dequeue, peek

4. Ponteiros e Alocação de Memória

Conceito: Variáveis que armazenam endereços de memória.

  • Uso: Alocação dinâmica (malloc, free), estruturas dinâmicas
  • Tipos: Ponteiro simples, ponteiro duplo
  • Regiões de memória: Stack (variáveis locais), Heap (alocação dinâmica)
